# Machine Health Monitoring Dashboard (Excel)

## Project Overview

This project presents an interactive **Machine Health Monitoring Dashboard** developed in **Microsoft Excel** using a predictive maintenance dataset.

The dashboard analyzes machine operational data and failure patterns to identify potential risks and support data-driven maintenance decisions. It demonstrates how Excel can be used to build analytical dashboards for industrial performance monitoring.

---

## Dashboard Features

* KPI cards for machine performance metrics
* Machine failure analysis by machine type
* Failure cause distribution analysis
* Tool wear analysis across machine types
* Temperature analysis for operational monitoring
* Interactive slicers for dynamic data filtering

---

## Tools & Technologies Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Analysis Techniques

---

## Dataset Description

The dataset contains machine operational data used for predictive maintenance analysis.

Key fields include:

* Machine Type
* Air Temperature
* Process Temperature
* Rotational Speed (RPM)
* Torque
* Tool Wear
* Failure Type
* Machine Failure Indicator

These variables help analyze how machine conditions relate to operational failures.

---

## Key Insights

* Certain machine types show higher failure rates compared to others.
* Increased tool wear is associated with a higher likelihood of machine failure.
* Temperature variations influence machine performance and reliability.

These insights demonstrate the importance of **predictive maintenance strategies** to reduce downtime and improve operational efficiency.
